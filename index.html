🎮 Gameplay Framework (Machina.Gameplay)
Reusable abstract building-blocks for any container-sorting puzzle game. Implement the abstract methods in your game-specific subclasses. All in Assets/Scripts/Machina/_Gameplay/.
ClassTypeResponsibility & Key Members
Item abstract Any discrete interactive entity (card, token, piece). State machine, IClickable, IPoolable. Tracks container refs (container, collectionPoint, storageTray). Override: Click(), OnPoolSpawn(), StateChanged().
ItemContainer abstract Stack/deck that holds Items. Owns GridCoordinate. Manages List<Item> items, selectable-top logic. Override: Click(), DestroySelf(bool).
CollectionPoint abstract Bus-like collection target with capacity and state machine. Override: TryCollect(), Capacity, DestroySelf().
StorageTray abstract Overflow receptacle. Tracks FreeSpace, fires CapacityChangedEvent. Override: CheckLoseCondition(), DestroySelf().
InputHandler abstract Raycast-driven input with AnyTapEvent. ToggleInput(bool) for enable/disable. Override: DetectInput(), DeselectItems(), TryGetRaycastHit().
MovementHandler abstract Undo cache stack (Stack<Move> moveCache). 3 abstract MoveItems() overloads for Item→Container, Item→StorageTray, Item→CollectionPoint.
ContainerSpawner abstract Grid occupancy tracking, world position calculation, new-slot search. Override: SpawnContainer(Vector2Int, Vector3).
ContainerHandler abstract Grid query API: EffectiveColCount, EffectiveRowCount, container lookup. Override: SpawnContainers(int, Action).
GridMath static Stateless grid utilities: ComputeGridDimensions(), ComputeWorldPosition(), GetBaseGridCoordinates(), GetExtraGridCoordinates().
Move record Concrete undo record: List<Item> Items, ItemContainer FromContainer, Vector2Int FromContainerCoordinate.
CustomFormationPreset ScriptableObject Grid cell matrix for custom container formation layouts. Odin-powered editor with toggleable cell matrix. GetActiveCoordinates() → List<Vector2Int>.

🔥 Live Ops Events (Machina.Events)
All live ops events inherit from GameFeature ScriptableObject. Registered in EventsManager via serialized fields. Concrete events are accessed via static properties: EventsManager.WinStreakEvent, EventsManager.LavaQuestEvent, etc.
ClassBaseKey Behaviour
GameFeature abstract MonoBehaviour Base for all features. Progress tracking, UI integration, IsRunning, IsVisible, IsInteractable. Override GetFeatureKey(), GetUnnyId().
TimedGameFeature abstract Adds countdown timer, start/end lifecycle hooks, OnEventStarted(), OnEventEnded(), stats persistence via ES3.
MilestonedGameFeature abstract Adds milestone list, claim logic, GetCurrentMilestone(), progress display.
MainLeagueEvent TimedGameFeature Competitive league season. Creates/ends LeagueGroup via LeagueManager. Increments score on level win. Exposes GetCurrentLeagueIcon(), GetPlayerRank().
WinStreakEvent TimedGameFeature Win streak multiplier bonus. Tracks consecutive wins, bonus multipliers with league bonus dict.
LavaQuestEvent TimedGameFeature Photo-map progression event with stage unlocking, avatar placement, and visual rewards.
MemoryMadnessEvent MilestonedGameFeature Memory card match mini-game embedded as a live op. Spawns MemoryCard grid, tracks match progress, milestone rewards.
PiggyBank MilestonedGameFeature Accumulating piggy bank. Fills from gameplay earnings; offers collect via IAP or RV.
Notification ScriptableObject Local/push notification definition: title, body, schedule delay. Used by NotificationsManager.
📐 Key Patterns & Conventions
Singleton Manager
[DefaultExecutionOrder(-80)] public class GameManager : Manager<GameManager> { protected override void OnAwake() { } protected override void AddListeners() { } protected override void OnDestroyCallback() { } }
Static UnityEvent
[Serializable] public class GameStateEvent : UnityEvent<GameState> { } public static GameStateEvent GameStateChangedEvent = new GameStateEvent();
Typed Game Cast
// In a GameController<T> subclass: private GameSortDash SortieGame => game as GameSortDash; public static GameSortDash Game => Instance?.SortieGame;
Serialized Properties
// Non-serialized state public LevelState State { get; private set; } // Serialized with read-only public access [SerializeField] private int lives; public int Lives => lives;
Logging
Machina.Logger.Log("message", this); Machina.Logger.LogWarning("warn", this); Machina.Logger.LogError("error", this);
Object Pooling
// Implement IPoolable on your MonoBehaviour: public void OnPoolSpawn() { } public void OnPoolDespawn() { } // Get and release via PoolManager: PoolManager.Get<MyItem>(); PoolManager.Release<MyItem>(item);
ConventionRule
BracesAllman style — opening brace always on new line. Required for ALL control flow bodies, no exceptions.
No varAlways explicit types: List<Card> list = new List<Card>();
Unity nullAlways == null / != null. Never is null / is not null (bypasses Unity equality).
RegionsEvery class: FIELDS AND PROPERTIES → EVENTS → PUBLIC API → PRIVATE
NamingClasses/Methods/Properties: PascalCase. Private fields: camelCase (no underscore). Constants: SCREAMING_SNAKE_CASE.
UI prefixAll UI classes start with lowercase u prefix after the UI prefix (e.g., UIPanel, uiPanel for private refs).
Early returnsPrefer guard clauses over deep nesting.
No hot-path LINQLINQ only if it doesn't allocate in Update/FixedUpdate loops.
